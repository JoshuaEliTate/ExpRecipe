# ExpRecipe
Easily search for recipes and save their ingredients into a grocery list.

## Description
This webpage was built utilizing javascript and two server side APIs in order to build a webpage that allows a user to search for recipes and add their ingredients into a grocery list (utilizing Edamam API and local storage) and search for nearby grocery stores utilizing the Google Maps API. 

## Installation
Webpage can be accessed using this link: https://joshuaelitate.github.io/ExpRecipe/
